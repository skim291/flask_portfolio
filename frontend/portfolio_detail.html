Flask 기반 웹 포트폴리오

Flask 프레임워크를 활용해 백엔드와 프런트엔드의 구조적 통합을 구현한 개인 포트폴리오 웹사이트입니다.

프로젝트 개요

프로젝트명: 김서현의 PORTFOLIO
개발기간: 2025.07 ~ 2025.08
사용 기술: Flask, Python, HTML5, CSS3, JavaScript, Jinja2

Flask의 라우팅, 템플릿 상속, 정적 파일 관리 구조를 이해하고 이를 기반으로 동적 웹 포트폴리오를 설계했습니다.

구현 과정

- 프로젝트 구조 설계

- app.py 라우팅 구성

- 템플릿 및 정적 리소스 관리

  • base.html: 공통 레이아웃
  • index.html: 소개 페이지
  • projects.html: 프로젝트 목록
  • contact.html: 연락처
