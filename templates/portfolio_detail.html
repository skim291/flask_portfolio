Flask 기반 웹 포트폴리오

Flask 프레임워크를 활용해 백엔드와 프런트엔드의 구조적 통합을 구현한 개인 포트폴리오 웹사이트입니다.

프로젝트 개요

Flask의 라우팅, 템플릿 상속, 정적 파일 관리 구조를 이해하고 이를 기반으로 웹 포트폴리오를 설계했습니다. 페이지 간 일관성을 유지하면서도 각기 다른 콘텐츠를 표현하도록 구성했습니다.

주요 구현

  • Flask 라우팅 기반 페이지 전환
  • 템플릿 상속 구조를 통한 재사용성 확보
  • 정적 파일 (CSS, JS, 이미지) 분리 관리
  • 카드형 프로젝트 UI 및 전환 효과

기술 스택

Flask Python HTML5 / CSS3 JavaScript Jinja2

결과 및 배운 점

Flask를 기반으로 백엔드와 프런트엔드의 연결 구조를 완전히 이해하게 되었으며,
유지보수성과 확장성을 모두 갖춘 개인 웹 애플리케이션을 완성했습니다.
